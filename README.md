# Fleet-Asset-Manager

An internal fleet management application for an Amazon DSP.

Built using Streamlit and Supabase, this tool provides:
1. Mobile-friendly daily vehicle audits
2. Real-time fleet tracking and dynamic alert mapping
3. Maintenance logging and mileage management

## Local Setup
```bash
pip install -r requirements.txt
```
To run the app locally:
```bash
streamlit run app.py
```
